How to Install on iOS

Follow these quick, easy steps to configure Delta on your iPhone or iPad.

1

Download the Configuration Profile

Tap the primary "Download Profile" button at the top of the page. Ensure you are using the Safari Browser for iOS systems to register it correctly.

2

Allow Profile Download Prompt

When iOS shows the notification popup stating: "This website is trying to download a configuration profile...", tap Allow.

3

Open iOS Settings

Launch your device's native Settings application and navigate directly to the "Profile Downloaded" menu item right near the top.

4

Install & Authenticate

Tap Install in the top-right corner, enter your passcode to authenticate the signature certificate, and tap Install once more to finish setting up Delta!
